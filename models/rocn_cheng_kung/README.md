# 成功級附加模型

本目錄保存不修改原版 Oliver Hazard Perry 長船體模型即可載入的成功級附加幾何。

目前包含：

- 1 至 7 號艦左右舷 40 mm/L70 機砲的懸臂平台、欄杆及斜撐。
- 成功級左右舷各一組 2×2 雄二／雄三混裝發射架，包含不同尺寸的箱體、加強肋、端蓋、甲板底座、承梁、A 字斜撐與設備箱。

遊戲 OBJ 採 Sea Power 的 1:100 比例，並沿用原版 `usn_parts` 材質。發射架配置依公開照片製作：每組兩枚雄二位於同一欄、兩枚雄三位於另一欄，總計全艦四枚雄二與四枚雄三。

重新產生並同步至模組資源：

```powershell
.\models\rocn_cheng_kung\generate_cheng_kung_additions.ps1
```

平台尺寸及支撐結構依公開照片估算；日後若取得造船圖或可量測的側視、俯視照片，可直接在產生器中微調。
